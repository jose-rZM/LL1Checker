Lexer functor for defining tokenization rules using Boost Spirit \doxylink{classLex}{Lex}. 

This struct is a functor that defines the tokenization rules for the lexer. It inherits from {\ttfamily boost\+::spirit\+::lex\+::lexer\texorpdfstring{$<$}{<}Lexer\texorpdfstring{$>$}{>}} and is used to configure the lexer with regular expressions and corresponding token IDs. The rules are added to the lexer during construction.
