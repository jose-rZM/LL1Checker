{\footnotesize\ttfamily void lexer\+::tokenize (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [private]}}



Loads and initializes necessary lexer symbols, then tokenizes input. 

This function dynamically loads the symbols "{}set\+\_\+yyin"{}, "{}yylex"{}, and "{}yylex\+\_\+destroy"{}. If any loading operation fails, the program aborts. Once symbols are loaded successfully, it tokenizes the input file using {\ttfamily yylex}. All tokens are stored in the {\ttfamily tokens\+\_\+} vector.

\begin{DoxyNote}{Note}
The function frees resources upon reaching the EOL (character value 1). This function is called only once for initial setup and tokenization. 
\end{DoxyNote}
